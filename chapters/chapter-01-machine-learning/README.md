## Chapter Overview

This chapter answers the most fundamental question in our entire ecosystem: **What is machine learning, and how is it different from traditional programming and from artificial intelligence?**

You will build your first models — a k-means clustering system for customer segmentation and a k-nearest neighbors classifier for flower species — while developing the mental models that every subsequent course depends on.

---

## Learning Objectives

By the end of this chapter, you will be able to:

1. Define machine learning and explain how it differs from rule-based programming
2. Distinguish supervised, unsupervised, and reinforcement learning
3. Explain the relationship between ML and AI
4. Implement k-means clustering for customer segmentation
5. Implement k-nearest neighbors (k-NN) for classification
6. Evaluate model performance with appropriate metrics
7. Identify and handle common data issues (outliers, missing values, scaling)

## Key Takeaways

- ML learns patterns from data instead of following explicit rules
- Supervised learning uses labeled examples; unsupervised finds hidden structure
- Algorithm choice depends on problem type, data size, and interpretability needs
- Data quality matters more than algorithm sophistication
