# Course 1 Resources

Supplementary materials for Applied ML & AI for Engineers.

## Cheat Sheets

- Scikit-learn estimator workflow: split, fit, predict, score, cross-validate
- Metric selection: accuracy, precision, recall, F1, ROC-AUC, MAE, RMSE
- Deployment checklist: serialization, schema validation, Docker, ONNX, monitoring

## Dataset Starters

- Iris, Titanic, MNIST, IMDB, and sample tabular datasets available through scikit-learn, Keras, or Kaggle
- For capstone work, prefer datasets with documented provenance, clear licenses, and stable download URLs

## Maintenance Notes

- Course 1 uses Keras/TensorFlow for neural-network chapters and PyTorch-first Hugging Face examples for current transformer fine-tuning.
- Azure chapters should be checked against current Azure AI service names and SDK versions before live use.
